# PulseLink User Manual

## Introduction

PulseLink is an emergency blood donor network that helps patients find compatible blood donors quickly during medical emergencies.

---

# User Roles

## Donor

A donor can:

* Register on the platform
* Update availability status
* Receive blood donation requests
* Accept or reject requests

## Patient

A patient can:

* Create blood requests
* Track request status
* View donor responses

---

# Donor Registration

Navigate to:

```text
Register as Donor
```

Provide:

* Full Name
* Phone Number
* Address
* Blood Group
* Last Donation Date

Submit the form to complete registration.

---

# Creating a Blood Request

Navigate to:

```text
Request Blood
```

Provide:

* Patient Name
* Phone Number
* Hospital Address
* Required Blood Group
* Emergency Notes

Click Submit Request.

---

# Donor Matching Process

When a request is submitted:

1. PulseLink searches for compatible blood donors.
2. Donor availability is verified.
3. Donation eligibility is checked.
4. Notifications are sent.
5. Donors can accept or reject requests.

---

# Request Status

Possible statuses:

## Pending

Request has been created.

## Accepted

A donor has accepted the request.

## Completed

Blood donation has been successfully completed.

## Cancelled

Request has been cancelled.

---

# Dashboard

The dashboard displays:

* Active Requests
* Accepted Requests
* Completed Requests
* Donor Information

---

# Eligibility Rules

A donor is eligible only if:

* The donor is available.
* The previous donation was at least 90 days ago.
