# Pulselink - Changelog

## [0.1.0] - 2026-06-09

### Added - POC Release

#### Backend
- User authentication system (register, login, refresh token)
- Donor management APIs
- Blood request creation and tracking
- Request response system for donor acceptance/decline
- SMS integration with Twilio
- JWT-based authorization with role-based access control
- Database models for all entities
- Error handling and validation middleware
- Rate limiting and security middleware
- SMS logging and tracking

#### Frontend
- Donor registration form
- Patient blood request form
- Donor dashboard for viewing requests
- Login and authentication
- Protected routes based on user roles
- Responsive UI design
- Real-time request updates (simulated)
- Authentication context for state management

#### Documentation
- Complete system specification
- Architecture documentation
- API testing guide
- Deployment guide
- Contributing guide
- Setup scripts
